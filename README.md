# ChatFlow AI Landing Page

A modern, responsive landing page built with React, TypeScript, Tailwind CSS, and GSAP animations.

## Color Palette

- **Primary**: `#E2852E`
- **Secondary**: `#F5C857`
- **Highlight**: `#FFEE91`
- **Accent**: `#ABE0F0`

Colors are configured in `tailwind.config.js` and can be used as Tailwind classes:
- `text-primary`, `bg-primary`, `border-primary`
- `text-secondary`, `bg-secondary`, `border-secondary`
- `text-highlight`, `bg-highlight`, `border-highlight`
- `text-accent`, `bg-accent`, `border-accent`

## Installation

```bash
npm install
```

## Development

